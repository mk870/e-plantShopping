# e-plantShopping

## 🌱 e Planting - Online Plant Shopping App

Welcome to **e PlantShopping**, an online plant shopping application designed to make buying plants easy, fast, and enjoyable. The project repository name is **e PlantShopping**.

This application allows users to browse different types of plants, view plant details, add items to their cart, and purchase plants online from the comfort of their homes.

---

## 🚀 Features

- Browse a variety of plants
- View detailed plant information
- Add plants to shopping cart
- Responsive and user-friendly interface
- Easy online shopping experience

---

## 🛠️ Technologies Used

- React
- JavaScript / TypeScript
- HTML5
- CSS3 / Styled Components
- Vite

---

## 📂 Repository

Repository Name: **e PlantShopping**

## 📸 Project Overview

The **e PlantShopping** app is built to provide plant lovers with a simple and modern platform for purchasing indoor and outdoor plants online. Users can explore plant categories, manage their shopping cart, and enjoy a smooth shopping experience.
